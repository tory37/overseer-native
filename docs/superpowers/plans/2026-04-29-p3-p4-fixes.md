# P3/P4 UI Polish & Infrastructure Fixes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement focused terminal indicators, per-session layout persistence, and fix the TypeScript Buffer global issue in the renderer.

**Architecture:** Extend the `Session` model with a `layout` object. Add a `session:update` IPC to persist layout changes. Refactor `useCompanion` hook to sync with the active session's layout. Switch to raw binary transfer for scrollback to avoid `Buffer` in the renderer.

**Tech Stack:** React, TypeScript, Electron, xterm.js, Zustand.

```typescript
interface SessionStore {
  // ...
  updateSession: (id: string, partial: Partial<Session>) => Promise<void>
}

export const useSessionStore = create<SessionStore>((set) => ({
  // ...
  updateSession: async (id: string, partial: Partial<Session>) => {
    if (!window.overseer?.updateSession) return
    await window.overseer.updateSession(id, partial)
    set(state => ({
      sessions: state.sessions.map(s => s.id === id ? { ...s, ...partial } : s)
    }))
  },
}))
```

### Task 4: Per-Session Layout Persistence Logic

**Files:**
- Modify: `src/renderer/hooks/useCompanion.ts`

- [ ] **Step 1: Sync state with activeSession layout**
Add `useEffect` to load layout when `activeSession` changes, and call `updateSession` when layout state changes.

```typescript
// src/renderer/hooks/useCompanion.ts

  // Load layout when session changes
  useEffect(() => {
    if (activeSession?.layout) {
      const { layout } = activeSession
      setSplitOpen(layout.splitOpen)
      setThreeWayOpen(layout.threeWayOpen)
      setSplitDirection(layout.splitDirection)
      setSplitSwapped(layout.splitSwapped)
      setSecondarySwapped(layout.secondarySwapped)
      setOuterSplitRatio(layout.outerSplitRatio)
      setInnerSplitRatio(layout.innerSplitRatio)
    } else {
      // Defaults if no layout persisted
      setSplitOpen(false)
      setThreeWayOpen(false)
      setSplitDirection('horizontal')
      setSplitSwapped(false)
      setSecondarySwapped(false)
      setOuterSplitRatio(0.5)
      setInnerSplitRatio(0.5)
    }
  }, [activeSession?.id])

  // Persist layout changes
  const persistLayout = useCallback(() => {
    if (!activeSession) return
    const layout: SessionLayout = {
      splitOpen,
      threeWayOpen,
      splitDirection,
      splitSwapped,
      secondarySwapped,
      outerSplitRatio,
      innerSplitRatio,
    }
    useSessionStore.getState().updateSession(activeSession.id, { layout })
  }, [activeSession?.id, splitOpen, threeWayOpen, splitDirection, splitSwapped, secondarySwapped, outerSplitRatio, innerSplitRatio])

  // Trigger persistence on changes (debounced or on dependency change)
  useEffect(() => {
    persistLayout()
  }, [splitOpen, threeWayOpen, splitDirection, splitSwapped, secondarySwapped, outerSplitRatio, innerSplitRatio])
```
