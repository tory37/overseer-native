# P1 Bugs & Terminal Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix P1 bugs (Copy/Paste, Split Breakage, Shortcut issues) and P2 polish (Settings toggle).

**Architecture:**
- **Clipboard/Context Menu:** Use Electron's `clipboard` and `Menu` APIs via IPC to ensure robustness. Add a custom context menu for terminals.
- **Split Management:** Refactor `onSplitClose` in `useCompanion` to correctly shift panes when closing intermediate ones. Update `TerminalPane` to be resilient to missing companion A.
- **Shortcuts:** Fix toggle logic in `App.tsx` and ensure shortcuts work when modals are open (but not capturing).

**Tech Stack:** React, TypeScript, Electron, xterm.js
